💿 VinylDesk 🎧
A high-performance, minimalist vinyl-style music visualizer for Lively Wallpaper.

VinylDesk is a glassmorphism-inspired interactive wallpaper designed for maximum aesthetic impact with a near-zero footprint. Built with pure optimization in mind, the entire project weighs only 35 KB.

✨ Key Features
🎵 Smart Media Detection: Fully compatible with Spotify, YouTube, and YouTube Music.

💿 Vinyl UI: Fluid LP-spinning animation with high-fidelity asset rendering.

🌈 Dynamic Aura: Real-time background blur that adapts to the current album's color palette.

⚡ Ultra-Lightweight: Engineered for minimal CPU/RAM usage.

🎧 Subtle Visualizer: A minimalist frequency bar design that breathes with the beat.

🚀 Installation
⚡ Easy Method (Recommended)
Open Lively Wallpaper.

Go to your Wallpaper Library.

Right-click any wallpaper → Select "Show in Explorer".

Go back one folder until you reach the wallpapers directory.

Extract the VinylDesk ZIP folder directly into this directory.

🧠 Manual Method
Extract the ZIP contents into:
C:\Users\%USERNAME%\AppData\Local\Lively Wallpaper\Library\wallpapers

⚠️ Important Notes
✔️ Structure Check: Your folder should look like this: VinylDesk/ containing index.html, LivelyInfo.json, and the js/ & styles/ folders.

❌ No ZIP loading: Lively will not recognize the file if it remains compressed. Please extract it first.
