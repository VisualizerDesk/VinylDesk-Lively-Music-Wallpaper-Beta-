VinylDesk 🎧

Minimal vinyl-style music wallpaper for Lively Wallpaper.

✨ Features
🎵 Detects current playing music
💿 Vinyl-style UI
🌈 Dynamic blurred background
⚡ Lightweight (low CPU usage)
🎧 Subtle visualizer
📸 Preview

Preview

🚀 Installation
⚡ Easy method (recommended)
Open Lively Wallpaper
Go to your wallpaper library
Right click → “Show in Explorer”
A folder like this will open:
C:\Users\USER\AppData\Local\Lively Wallpaper\Library\SaveData\wptmp\dybe4ky1.ufy
Go back one folder until you reach wallpapers
Extract the VinylDesk ZIP there
🧠 Manual method

Extract the ZIP directly into:

C:\Users\USER\AppData\Local\Lively Wallpaper\Library\wallpapers
⚠️ Important
❌ Do NOT use the .zip file without extracting
✔️ It should look like this:
VinylDesk/
 ├── index.html
 ├── LivelyInfo.json
 ├── js/
 ├── styles/
⚠️ Notes
Visualizer is simulated (low resource design)
Works with system media detection
